# Calculadora em Java

Projeto desenvolvido para praticar lógica de programação e conceitos básicos de Java.

## Funcionalidades
- Adição
- Subtração
- Multiplicação
- Divisão (com validação de divisão por zero)
- Potenciação
- Raiz quadrada

## Tecnologias
- Java
- Scanner
- Math (pow, sqrt)

## Aprendizados
- Estrutura switch-case
- Entrada de dados com Scanner
- Validação de entradas
- Boas práticas iniciais em Java
